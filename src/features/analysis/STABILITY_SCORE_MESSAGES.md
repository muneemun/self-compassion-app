# 📊 안정성 지표 문구 가이드

이 문서는 관계 튜닝 센터의 안정성 지표에서 사용되는 모든 동적 문구와 Zone 정보를 정리한 참고 자료입니다.

---

## 1. Zone 정보 정의

| Zone | 명칭 | 목표 범위 | 설명 |
|------|------|-----------|------|
| Zone 1 | 안전기지 | 15-25% | 가장 신뢰하고 의지하는 핵심 관계망 |
| Zone 2 | 심리적 우군 | 15-25% | 정서적 지지와 공감을 주고받는 관계 |
| Zone 3 | 전략적 동행 | 20-30% | 목표 지향적 협력 관계 |
| Zone 4 | 사회적 지인 | 10-20% | 적절한 거리를 유지하는 사회적 네트워크 |
| Zone 5 | 배경 소음 | 10-20% | 최소한의 에너지만 투입하는 관계 |

---

## 2. 안정성 점수 구간별 라벨 및 컬러

| 점수 범위 | 라벨 | 컬러 코드 | 의미 |
|-----------|------|-----------|------|
| 85% ~ 100% | 최적 | #4A5D4E (다크 그린) | 모든 구역이 이상적으로 균형 잡힌 상태 |
| 60% ~ 84% | 양호 | #7BA67E (라이트 그린) | 전반적으로 안정적이나 약간의 개선 여지 |
| 40% ~ 59% | 주의 | #E9A15A (오렌지) | 특정 구역에 불균형이 감지되어 조율 필요 |
| 0% ~ 39% | 위험 | #D98B73 (테라코타) | 심각한 불균형으로 정서적 소모가 큰 상태 |

---

## 3. 안정성 배지 하단 설명 문구 (동적 생성)

### 3.1 최적 상태 (85% ~)
**기본 문구:**
```
모든 구역의 에너지가 이상적으로 분배되어 있습니다.
```

### 3.2 양호 상태 (60% ~ 84%)
**불균형 Zone이 있는 경우:**
```
Zone {번호}({Zone명})에 약간의 편차가 있으나 전반적으로 안정적입니다.
```
**예시:** `Zone 1(안전기지)에 약간의 편차가 있으나 전반적으로 안정적입니다.`

**불균형 Zone이 없는 경우:**
```
전반적으로 안정적인 에너지 흐름을 보이고 있습니다.
```

### 3.3 주의 상태 (40% ~ 59%)
**불균형 Zone이 있는 경우:**
```
Zone {번호}({Zone명})에 {상태} 상태입니다.
```
- **{상태}**: `에너지 과다` 또는 `에너지 부족`
- **예시:** `Zone 1(안전기지)에 에너지 과다 상태입니다.`

**불균형 Zone이 없는 경우:**
```
일부 구역에 에너지가 쏠려있어 조율이 권장됩니다.
```

### 3.4 위험 상태 (0% ~ 39%)
**불균형 Zone이 있는 경우 (최대 2개 표시):**
```
{Zone 목록}의 심각한 불균형이 감지되었습니다.
```
**예시:** `Zone 1(안전기지), Zone 3(전략적 동행)의 심각한 불균형이 감지되었습니다.`

**불균형 Zone이 없는 경우:**
```
관계망의 불균형이 심화되어 정서적 소모가 큽니다.
```

## 4. 팝업 내 점수 구간별 설명 문구

### 최적 (85% ~)
```
모든 구역의 에너지가 이상적으로 분배되어 있습니다.
```

### 양호 (60% ~ 84%)
```
전반적으로 안정적인 에너지 흐름을 보입니다.
```

### 주의 (40% ~ 59%)
```
특정 구역에 에너지가 편중되어 조율이 필요합니다.
```

### 위험 (0% ~ 39%)
```
심각한 불균형으로 정서적 소모가 큰 상태입니다.
```

---

## 5. Zone별 에너지 분포 차트 표시 형식

각 Zone은 다음 형식으로 표시됩니다:

```
Zone {번호} {Zone명}
[━━━━━━━━━━] {현재%}%
목표: {최소}-{최대}%
```

**예시:**
```
Zone 1 안전기지
[━━━━━━━━━━] 35%  ⚠️ (목표 범위 초과)
목표: 15-25%
```

**색상 규칙:**
- 목표 범위 내: #4A5D4E (다크 그린)
- 목표 범위 초과/미달: #D98B73 (테라코타, 굵은 글씨)

## 6. 사용 예시 시나리오

### 시나리오 A: 안전기지 과다 투입
- **안정성 점수:** 43%
- **불균형 Zone:** Zone 1 (35%, 목표 15-25%)
- **배지 설명:** `Zone 1(안전기지)에 에너지 과다 상태입니다.`
- **팝업 차트:** Zone 1의 바가 빨간색으로 강조, 35% 표시

### 시나리오 B: 여러 구역 불균형
- **안정성 점수:** 28%
- **불균형 Zone:** Zone 1 (40%), Zone 3 (5%)
- **배지 설명:** `Zone 1(안전기지), Zone 3(전략적 동행)의 심각한 불균형이 감지되었습니다.`
- **팝업 차트:** 두 Zone 모두 빨간색 강조

### 시나리오 C: 이상적 균형
- **안정성 점수:** 92%
- **불균형 Zone:** 없음
- **배지 설명:** `모든 구역의 에너지가 이상적으로 분배되어 있습니다.`
- **팝업 차트:** 모든 Zone이 초록색

## 7. 문구 톤 앤 매너 가이드

### 원칙
1. **명확성**: 사용자가 즉시 이해할 수 있는 구체적 정보 제공
2. **중립성**: 판단보다는 현상 설명에 집중
3. **행동 유도**: 문제가 있을 경우 '조율', '관리' 등의 액션 제안
4. **따뜻함**: '위험', '긴급' 같은 강한 표현 대신 '주의', '권장' 사용

### 금지 표현
- ❌ "당신의 관계가 위험합니다"
- ❌ "즉시 조치가 필요합니다"
- ❌ "심각한 문제가 발생했습니다"

### 권장 표현
- ✅ "조율이 권장됩니다"
- ✅ "에너지 분배를 재조정해보세요"
- ✅ "약간의 편차가 감지되었습니다"
