# FT-11 집중 조율 결과: 관계 패턴 분석 로직 (Pattern Analysis Engine)

이 문서는 집중 조율(Tournament) 완료 후, 사용자의 무의식적 선택 데이터를 바탕으로 도출되는 **7가지 관계 패턴**의 판정 로직과 심리적 해설 가이드를 정의합니다.

---

## 1. 분석 입력 데이터 (Input Data)

분석 엔진은 다음 두 가지 핵심 변수를 조합하여 패턴을 판정합니다.
1.  **진입 맥락 (Entry Lens)**: 조율을 시작할 때 선택한 렌즈 (`Positive`, `Negative`, `Frequency`, `None`)
2.  **선택 분포 (Zone Distribution)**: 최종 승리자(Winners)들이 속한 궤도 구역(Zone 1~5) 비율

---

## 2. 7가지 핵심 패턴 정의

### 2.1 맥락 특화 패턴 (Context-Specific)

| 패턴명 | 판정 로직 (Logic) | 심리적 키워드 | 해설 가이드 (Insight) |
| :--- | :--- | :--- | :--- |
| **🛡️ 경계 (Boundaries)** | `Entry Lens === 'Negative'` | 자기 보호, 방어선 | 가장 소모적인 관계들 사이에서 심리적 방어선을 구축하는 선택을 했습니다. 나를 지키기 위한 건강한 거리두기가 핵심입니다. |
| **🕒 정제 (Efficiency)** | `Entry Lens === 'Frequency'` | 시간 가치, 우선순위 | 가장 많은 시간을 점유한 관계들을 조율하며 시간의 밀도를 재평가했습니다. 의미 있는 교감에 집중하려는 욕구가 보입니다. |

### 2.2 성향 기반 패턴 (Disposition-Based)
*판정 조건: 위 맥락 패턴에 해당하지 않으며, 특정 Zone의 비중이 70% 이상일 때*

| 패턴명 | 판정 로직 (Logic) | 핵심 가치 | 해설 가이드 (Insight) |
| :--- | :--- | :--- | :--- |
| **✨ 안정 (Stability)** | `Winner Zone 1 Count ≧ 70%` | 핵심 그룹, 신뢰 | 새로운 자극보다 나를 지탱해주는 [깊은 안정감]을 선택했습니다. 핵심 그룹을 보호하려는 무의식이 작동하고 있습니다. |
| **🌱 성장 (Growth)** | `Winner Zone 2 Count ≧ 70%` | 영감, 시너지 | 당신의 마음은 [성장]을 향해 기울어 있습니다. 가치관을 공유하는 정서적 공유 그룹과의 교감이 당신의 활력원이 되고 있습니다. |
| **🔥 활력 (Vitality)** | `Winner Zone 3 Count ≧ 70%` | 목표, 추진력 | 명확한 [목표와 활력]을 주는 기능적 협력 관계에 집중하고 있습니다. 현재 추진 중인 일이나 변화에 에너지가 실리고 있는 상태입니다. |
| **🌌 확장 (Expansion)** | `Winner Zone 4/5 Count ≧ 70%` | 호기심, 외연 | 익숙함보다 [새로운 연결과 외연]에 더 민감하게 반응하고 있습니다. 단순 인지 관계의 확장을 통해 새로운 에너지를 수혈받고 싶은 상태입니다. |

### 2.3 조화 패턴 (General)

| 패턴명 | 판정 로직 (Logic) | 핵심 가치 | 해설 가이드 (Insight) |
| :--- | :--- | :--- | :--- |
| **⚖️ 균형 (Harmony)** | 위 모든 조건에 해당하지 않음 | 유연함, 다양성 | 각기 다른 역할을 하는 관계들 사이에서 [조화로운 균형]을 선택했습니다. 치우치지 않는 유연한 자아 상태를 유지하고 있습니다. |

## 3. UI/UX 시각화 프로필

패턴별로 결과 화면(FT-11)의 `Summary Insight` 섹션에 적용되는 시각 요소입니다.

| 패턴 | 테마 컬러 | 아이콘 (Lucide) | 시각적 메타포 |
| :--- | :--- | :--- | :--- |
| **안정** | `#4A5D4E` | `Shield` | 나를 감싸는 방패 |
| **성장** | `#D98B73` | `TrendingUp` | 위로 뻗어가는 새싹 |
| **활력** | `#D4AF37` | `Zap` | 번쩍이는 에너지 카드 |
| **경계** | `#737874` | `Shield` | 단단한 돌담벽 |
| **정제** | `#6B8E23` | `History` | 돌아가는 모래시계 |
| **확장** | `#C0C0C0` | `Sparkles` | 퍼져나가는 별빛 |
| **균형** | `Primary` | `Heart` | 뛰고 있는 심장 |

---

## 4. 데이터 반영 (Feedback Loop)

패턴 분석 결과는 다음 데이터에 영향을 미칩니다.
1.  **안정성 점수(Stability Score)**: 조율 결과에 따라 `+N%` 보너스 부여
2.  **관계 온도 업데이트**: 승리한 인물들의 `temperature` 소폭 상승 보정
3.  **자아 리포트**: '나의 무의식적 관계 지향성' 통계 데이터로 축적
