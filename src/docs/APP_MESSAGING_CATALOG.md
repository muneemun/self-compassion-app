# Self-Compassion App Messaging Catalog

이 문서는 서비스 전반에 사용되는 팝업(Alert), 모달(Modal), 알림(Reminder) 및 가이드 문구를 체계적으로 정리한 카탈로그입니다. 각 메시지 확인 후 이동하는 목적지 화면을 함께 정의하여 사용자 경험의 연속성을 관리합니다.

---

## 1. 알림 및 리마인더 (Push & In-app Notifications)
사용자의 성찰을 유도하고 분석된 가치를 배달하며, 클릭 시 해당 기능 화면으로 즉시 연결합니다.

### A. 기록 리마인더 (Action-Oriented)
| 항목명 | 실제 알림/팝업 메시지 (Nudge) | 이동 화면 (Destination) |
| :--- | :--- | :--- |
| **오늘 기록 리마인더** | "오늘 당신의 감정 온도는 몇 도였나요? 마음이 머문 행성을 기록해 보세요." | **Main Orbit Map** (또는 기록 추가 팝업) |
| **정기 궤도 점검** | "행성들의 중력이 변하고 있어요. 관계 지도를 재배치하여 균형을 맞출 시간입니다." | **Relationship Tuning Dashboard** |

### B. 분석 리포트 알림 (Insight-Oriented)
| 항목명 | 실제 알림/팝업 메시지 (Insight) | 이동 화면 (Destination) |
| :--- | :--- | :--- |
| **개인 건강 리포트** | "지난주 마음 날씨가 분석되었습니다. 당신이 가장 평온했던 순간을 확인해 보세요." | **Self Health Report** |
| **관계 균형 리포트** | "소셜 우주의 새로운 균형점이 발견되었습니다. 변화된 궤도 리포트가 도착했습니다." | **Relationship Report** |

## 2. 시스템 알럿 및 경고 (Alerts & Warnings)
사용자의 주요 액션에 대해 확인을 받거나 오류를 안내하며, 확인 후 이전 화면으로 돌아가거나 다음 단계로 진행합니다.

### A. 입력 및 유효성 검사 (Validation)
- **이름 미입력**: `[알림]` "이름을 입력해주세요." → (현재 화면 유지)
- **관계 유형 미선택**: `[알림]` "관계 유형을 입력해주세요." → (현재 화면 유지)
- **활동 내용 미입력**: `[내용 입력]` "어떤 활동을 했는지 주제를 적어주세요." → (현재 화면 유지)
- **분석 데이터 부족**: `[분석 데이터 부족]` "조율을 진행하기 위한 관계 데이터가 충분하지 않습니다." → **Relationship Tuning Dashboard**

### B. 파괴적 액션 및 중단 (Destructive & Cancel)
- **진단 중단**: `[테스트 중단]` "지금 나가면 테스트 결과가 저장되지 않습니다. 정말 나갈까요?" → **Relationship Detail** (또는 메인 맵)
- **관계 삭제**: `[관계 삭제]` "'{이름}'님과의 모든 관계 데이터가 삭제됩니다. 정말 삭제하시겠습니까?" → **Main Orbit Map**
- **프로필 이미지 선택 취소**: (시스템 기본 알럿) → (현재 화면 유지)

### C. 성공 및 완료 (Success)
- **정보 업데이트 (내 프로필)**: `[저장 완료]` "프로필 정보가 업데이트되었습니다." → **Settings Screen**
- **정보 업데이트 (인맥 프로필)**: `[성공]` "프로필 정보가 업데이트되었습니다." → **Relationship Detail**
- **관계 추가 완료**: (자동 전환) → **Relationship Diagnosis** (초기 진단)

---

## 3. 시스템 권한 요청 (Permissions)
기능 수행을 위해 필요한 접근 권한 안내 문구입니다.

- **갤러리/카메라 접근 권한**: 권한 허용 시 **이미지 선택기/카메라** 호출, 거부 시 **현재 화면 유지**
- **연락처 접근 권한**: 권한 허용 시 **연락처 목록 추출**, 거부 시 **Relationship Entry**

---

## 4. 가이드 및 정보 모달 (Information & Guides)
설명 확인 후 '확인' 또는 'X' 버튼을 통해 **기존에 머물던 분석/상세 화면**으로 복귀합니다.

- **관계 지표 (안정성, 온도 등)** 확인 → **Relationship Detail / Report**
- **오빗 존 (Zone 1~5)** 확인 → **Relationship Detail / Main Map**

---

## 5. 대화형 인터페이스 (Interactive Dialogs)
사용자의 선택에 따라 분기되는 화면 경로를 정의합니다.

### A. 건강 확인 브릿지 (Relationship Health Check)
- **질문**: "어떤 진단을 다시 진행하시겠습니까?"
- **선택지에 따른 이동**:
    - `오빗 존(Zone) 재설정` → **Relationship Diagnosis (ZONE 모드)**
    - `질적 캐릭터 심화 진단` → **Relationship Diagnosis (RQS 모드)**
    - `취소` → **Relationship Detail** (현재 화면 유지)

### B. 프로필 관리 메뉴 (More Options)
- **질문**: "수행할 작업을 선택하세요."
- **선택지에 따른 이동**:
    - `프로필 정보 관리` → **Relationship Profile**
    - `진단 기록 보기` → **Relationship Report**
    - `취소` → **Relationship Detail** (현재 화면 유지)
